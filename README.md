# CNN-helmet-detection

# Overview
This project is a computer vision-based helmet detection system designed to identify whether individuals are wearing safety helmets on road and construction site. The solution leverages Convolutional Neural Networks (CNN) to automatically analyse visual data and improve safety compliance monitoring.



# Features
- Detects helmet usage in images
- Built using deep learning (CNN)
- Supports safety monitoring in real-world environments
- Scalable for integration into surveillance systems

---

# Technologies Used
- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Matplotlib


# Results
Achieved reliable detection accuracy of 0.90 through data preprocessing and model tuning
Demonstrates practical application of AI in health and safety monitoring
